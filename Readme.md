
### npm安装
```javascript
npm i waga-cli
```
### 模板添加
```javascript
waga add
```
### 模板列表
```javascript
waga list
```
### 模板删除
```javascript
waga del
```
### 模板初始化
```javascript
waga init
```
### 注意事项
- 1，如果安装时不成功，可npm install waga-cli --force强制安装
- 2，安装后，需要waga add配置模板信息
- 3，模板信息配置成功后，可waga init初始化模板创建项目
- 4，模板可为空，通过waga list可查看，waga del可删除
- 附：模板即脚手架中要用到的工具、环境和配置，需要根据自己的需求去上传git，实现脚手架的健全
